### 3.3 HTTPS/安全性

#### 当前状态

| 安全项 | 状态 | 说明 |
|--------|------|------|
| HTTPS | ✅ 已启用 | https://leeknives.com/ |
| SSL 证书 | ✅ 已部署 | 需检查有效期 |
| 隐私政策 | ✅ 已发布 | 内容完整 |
| 混合内容 | ⚠️ 待检查 | 可能存在 HTTP 资源 |

#### 安全检查清单

```
□ SSL 证书有效期检查（建议至少 6 个月）
□ 检查是否存在混合内容警告
□ 实施 HSTS（HTTP Strict Transport Security）
□ 添加安全响应头：
  • X-Frame-Options: DENY
  • X-Content-Type-Options: nosniff
  • X-XSS-Protection: 1; mode=block
  • Content-Security-Policy: [自定义策略]
□ 确保所有外部资源使用 HTTPS
□ 实施 HTTPS 强制跳转（301）
```

#### 推荐安全头信息配置

```apache
# HSTS - 强制 HTTPS
Header always set Strict-Transport-Security "max-age=31536000; includeSubDomains"

# 防止点击劫持
Header always set X-Frame-Options "DENY"

# 防止 MIME 类型嗅探
Header always set X-Content-Type-Options "nosniff"

# XSS 防护
Header always set X-XSS-Protection "1; mode=block"

# 内容安全策略（示例）
Header always set Content-Security-Policy "default-src 'self'; script-src 'self' 'unsafe-inline'; style-src 'self' 'unsafe-inline'; img-src 'self' data: https:;"
```

### 4.4 图片 ALT 标签

#### 重要性说明

| 作用 | 说明 |
|------|------|
| SEO | 帮助搜索引擎理解图片内容 |
| 无障碍 | 屏幕阅读器读取给视障用户 |
| 备用文本 | 图片加载失败时显示 |
| 图片搜索 | 提升 Google 图片搜索排名 |

#### ALT 标签编写指南

```
✅ 优秀 ALT 标签：
• 描述性：准确描述图片内容
• 简洁：通常 10-15 词
• 含关键词：自然融入，不堆砌
• 独特：每张图片不同

❌ 避免：
• 过于简单：alt="knife"
• 关键词堆砌：alt="knife wholesale knife OEM knife China"
• 重复：多张图片相同 ALT
• 包含"图片 of"：冗余
```

#### ALT 标签示例

| 图片类型 | ❌ 错误示例 | ✅ 正确示例 |
|----------|------------|------------|
| 产品图 | `alt="knife"` | `alt="8-inch Damascus steel Japanese chef knife"` |
| 服务图 | `alt="service"` | `alt="OEM knife custom design and development process"` |
