# 外链建设 Skill

## 概述

制定系统化的外链建设策略，分析竞品外链来源，识别高质量外链机会，
并提供可执行的外链获取方案和 outreach 模板。

## 前置条件

开始前确认：
- [ ] 目标网站 URL
- [ ] 主要竞品网站（1-3 个）
- [ ] 目标市场/地区
- [ ] 外链建设预算（如有）
- [ ] 当前外链状况简述（可选）

## 核心工作流

### 步骤 1：外链质量评估标准

读取 `references/link-quality-criteria.md`，按以下维度评估：

**域名权威度（25 分）**
- DR/DA ≥ 70: 25 分
- DR/DA 50-69: 20 分
- DR/DA 30-49: 15 分
- DR/DA < 30: 5 分

**相关性（25 分）**
- 同行业/同品类：25 分
- 相关行业：15 分
- 无关行业：5 分

**流量质量（20 分）**
- 月流量 ≥ 10 万：20 分
- 月流量 1-10 万：15 分
- 月流量 < 1 万：5 分

**外链自然度（15 分）**
- 外链增长曲线自然：15 分
- 有可疑增长峰值：5 分

**内容质量（15 分）**
- 原创深度内容：15 分
- 一般内容：8 分
- 低质/采集：0 分

**综合评级：**
- 80-100 分：A 级（优先获取）
- 60-79 分：B 级（值得争取）
- 40-59 分：C 级（酌情考虑）
- < 40 分：D 级（避免）

### 步骤 2：竞品外链分析

分析竞品外链来源，分类统计：

| 外链类型 | 描述 | 获取难度 |
|----------|------|----------|
| 客座博客 | 在行业博客发布文章 | 中 |
| 新闻稿 | 媒体报道/新闻网站 | 高 |
| 资源页面 | 行业资源列表/推荐页 | 低 - 中 |
| 论坛/社区 | Reddit、知乎、行业论坛 | 低 |
| 目录网站 | 行业目录/黄页 | 低 |
| 合作伙伴 | 供应商/客户/合作伙伴 | 低 |
| 社交媒体 | 社交信号 | 低 |
| HARO/问答 | 专家引用/问答平台 | 中 |

### 步骤 3：外链机会识别

基于竞品分析和行业特点，识别外链机会：

**低垂果实（优先）**
- 竞品有但我方没有的外链
- 资源页面提交
- 行业目录收录
- 未认领的品牌提及

**中等难度**
- 客座博客投稿
- 专家引用（HARO）
- 合作伙伴交换

**高难度**
- 权威媒体报道
- 行业研究报告引用
- 教育机构链接（.edu）

### 步骤 4：外链获取策略

为每类机会制定具体行动方案：

```
## 外链机会：[名称]

**目标网站：** [URL]
**外链类型：** [类型]
**预估难度：** [低/中/高]
**预估价值：** [A/B/C/D]

**获取方法：**
1. [具体步骤 1]
2. [具体步骤 2]
3. [具体步骤 3]

**所需资源：**
- [时间/人力/预算]

**Outreach 模板：**
[参考 references/outreach-templates.md]
```

## 错误处理

- 无法获取竞品数据 → 建议使用公开工具（如 Ahrefs Webmaster Tools 免费版）
- 外链机会过少 → 扩大竞品范围或调整行业定义
- 预算有限 → 优先推荐免费/低成本外链策略

## 风险提示

**避免的外链类型：**
- PBN（私有博客网络）
- 外链农场
- 自动化外链工具
- 低质目录网站
- 评论垃圾外链

**Google 算法风险：**
- Penguin 算法：打击垃圾外链
- 不自然外链增长可能触发人工审核
