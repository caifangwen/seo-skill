# 品牌 SEO 规范 Skill（内部）

## 概述

确保所有 SEO 内容符合公司品牌规范，包括品牌词使用、语调风格、
视觉元素描述、合规声明等。

## 前置条件

- [ ] 待检查内容
- [ ] 目标发布渠道（官网/博客/第三方平台）
- [ ] 目标受众（企业客户/个人用户/开发者）

## 核心检查项

### 1. 品牌词使用规范

**公司/产品名称：**
- 正确：`[ProductName]`（首字母大写）
- 错误：`[productname]`, `[PRODUCTNAME]`

**首次提及规则：**
- 首次提及必须使用全称：`[公司全称]（以下简称"[简称]"）`
- 后续可使用简称

**竞品提及规则：**
- 保持客观中立语气
- 不使用贬义形容词
- 使用"竞品名称"而非"某竞品"

---

### 2. 语调风格

**目标语调：** 专业、友好、可信赖

**避免：**
- ❌ 过于销售化语言（"最好"、"第一"、"领先"）
- ❌ 绝对化表述（"绝对"、"保证"、"100%"）
- ❌ 技术黑话堆砌

**推荐：**
- ✅ 数据支撑的表述（"提升 30% 效率"）
- ✅ 客户案例佐证
- ✅ 清晰易懂的解释

---

### 3. 合规声明

**必须包含的声明：**

| 内容类型 | 必需声明 |
|----------|----------|
| 产品功能介绍 | "功能可能因版本而异" |
| 数据/效果展示 | "实际效果因使用情况而异" |
| 竞品对比 | "数据来源于公开渠道，仅供参考" |
| 客户案例 | "已获得客户授权" |

---

### 4. 视觉元素描述

**图片/图表说明：**
- 必须包含 alt 文本
- 图表需注明数据来源
- 截图需标注版本号和日期

**品牌色使用：**
- 主色：`[色值]`
- 辅助色：`[色值]`
- 警示色：`[色值]`

---

## 检查流程

### 步骤 1：内容扫描

识别以下内容元素：
- 品牌词出现位置和形式
- 数据/效果声称
- 竞品提及
- 合规声明存在性

### 步骤 2：规范比对

逐项检查：
- [ ] 品牌词格式正确
- [ ] 首次提及使用全称
- [ ] 语调符合品牌风格
- [ ] 数据声称有来源
- [ ] 合规声明完整

### 步骤 3：问题标注

格式：
```
**位置：** 第 X 段
**问题：** [描述]
**建议修改：** [修改后文本]
**严重程度：** 高/中/低
```

### 步骤 4：输出检查报告

```
## 品牌规范检查报告

**检查内容：** [标题/URL]
**检查日期：** [日期]

### 概览
✅ 通过项：X 项
⚠️ 需修改：X 项
❌ 严重问题：X 项

### 详细问题

[逐项列出]

### 修改后版本

[提供完整修改后内容]
```
