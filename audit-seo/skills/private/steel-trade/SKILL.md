# Steel Trade Domain Skills

## 概述

钢铁外贸 B2B SEO 专用 Skill 集合，包含 5 个核心 Skill，覆盖关键词研究、内容优化、技术审计、外链建设和报告生成的完整工作流。

这是专为钢铁外贸行业设计的**领域特定 Skill 体系**，基于通用 SEO 技能之上构建，针对钢铁行业特点进行了深度定制。

---

## 核心特点

### 钢铁外贸 B2B SEO 特点

1. **买家市场特定**：中东、东南亚、非洲、南美（而非欧美）
2. **产品规格为核心**：规格 + 标准 + 用途 三要素关键词模型
3. **询盘转化为唯一目标**：流量本身无意义，询盘才是核心
4. **竞品差异化**：印度、土耳其、韩国钢厂为主要竞品

### 核心逻辑

> **"从产品规格关键词到买家询盘"**

所有 5 个 Skill 都围绕这一核心链条展开。

## 工作流串联

### 新产品上线工作流

```
steel-keyword-research → steel-content-optimizer → steel-report-generator
```

1. **steel-keyword-research**: 挖掘新产品关键词，分析目标市场
2. **steel-content-optimizer**: 优化产品页面，确保询盘转化
3. **steel-report-generator**: 追踪新产品页面的询盘表现

---

### 网站优化工作流

```
steel-technical-audit → steel-keyword-research → steel-content-optimizer → steel-link-building → steel-report-generator
```

1. **steel-technical-audit**: 诊断网站技术问题
2. **steel-keyword-research**: 分析关键词机会
3. **steel-content-optimizer**: 优化核心页面
4. **steel-link-building**: 建设行业外链
5. **steel-report-generator**: 生成优化效果报告

---

### 询盘下降排查工作流

```
steel-technical-audit → steel-content-optimizer → steel-report-generator
```

1. **steel-technical-audit**: 检查询盘表单功能、技术配置
2. **steel-content-optimizer**: 优化询盘转化路径
3. **steel-report-generator**: 分析询盘数据，找出下降原因

## 触发规则

### 优先触发 steel-* Skill 的场景

当用户输入包含以下关键词时，优先触发钢铁外贸 Skill 而非通用 SEO Skill：

| 关键词类型 | 示例 | 触发 Skill |
|------------|------|------------|
| 钢铁产品 | rebar, HRC, steel pipe, galvanized coil | steel-* |
| 钢铁标准 | ASTM A615, SS400, SPHC, DX51D | steel-* |
| 询盘相关 | inquiry, RFQ, quote, buyer | steel-* |
| 买家国家 | Saudi Arabia, Vietnam, Nigeria, Brazil | steel-* |
| 钢铁行业 | steel industry, B2B steel, steel export | steel-* |

### 触发优先级

1. **明确钢铁产品 + SEO 需求** → steel-* Skill
2. **通用 SEO 需求** → public/* Skill
3. **模糊需求** → 询问用户是否为钢铁外贸网站
