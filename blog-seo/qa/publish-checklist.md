# 发布前完整检查清单

**执行时机**: SEO 检查（`seo-checklist.md`）完成后，正式发布前  
**用途**: 确保文章内容完整、图片真实、链接有效、用户体验良好

---

## ① 内容完整性

```markdown
- [ ] Front Matter 完整（title、date、categories、coverImage）
- [ ] 开篇 2-3 段，包含 1-2 个内部链接
- [ ] 每个主要章节有图片（至少每 500 字 1 张）
- [ ] 包含优缺点平衡分析
- [ ] 有对比章节（如适用）
- [ ] 结论段落包含 CTA 和 request-a-quote 链接
- [ ] 段落长度适中（3-5 句/段）
- [ ] 标题层级正确（## 主章节，### 子章节）
- [ ] 列表格式统一
- [ ] 无拼写和语法错误
- [ ] 使用第二人称 "you/your"
```

---

## ② 图片资源验证 ⚠️

```markdown
- [ ] 所有图片链接来自网站现有资源或已下载到本地
- [ ] 未虚构不存在的图片文件名
- [ ] 图片年份优先使用 2026/2025 年资源
- [ ] 避免使用早于 2022 年的图片（除非必要）
- [ ] 图片描述与内容相关 ⭐ 重要
- [ ] 每章节至少有 1 张配图
- [ ] 图片文件名符合规范（小写 + 连字符）
- [ ] 图片已上传到正确路径（/wp-content/uploads/YYYY/MM/）
```

**图片相关性检查**（钢材文章）:
```markdown
- [ ] 开篇图片：显示刀具/钢材特写，非随机金属图
- [ ] 硬度章节：硬度测试相关图片，非健身房/钻石图
- [ ] 耐腐蚀章节：不锈钢/防水场景，非生锈金属图
- [ ] 边缘保持：锋利刀刃特写，非钝刀/破损刀图
- [ ] 打磨章节：磨刀石/打磨工具，非电动砂轮机
- [ ] 使用场景：与章节描述匹配（厨房/EDC/户外）
- [ ] 避免使用 Picsum 随机种子图片
```

---

## ③ 内部链接检查

```markdown
- [ ] 内部链接密度适当（每 300-400 字 1 个）
- [ ] 锚文本为描述性关键词，非"点击这里"
- [ ] 结尾包含 /request-a-quote/ 链接
- [ ] 所有内部链接 URL 有效，无 404
```

---

## ④ SEO 元素（快速核查）

```markdown
- [ ] Title 含核心关键词，长度 50-60 字符
- [ ] 开篇 100 词内出现核心关键词
- [ ] 至少 1 个 H2 含核心关键词变体
- [ ] 图片 Alt 属性描述性且含关键词
```

> 完整 SEO 检查见 `qa/seo-checklist.md`

---

## ⑤ 用户体验

```markdown
- [ ] 文章结构清晰，长文章有目录导航
- [ ] 对比表格易于阅读，列宽合适
- [ ] CTA 位置明显且自然（不突兀）
- [ ] 段落不过长，移动端友好（每段不超过 5 句）
```

---

## ⑥ 发布信息确认

```markdown
- [ ] date 字段使用 2026 年
- [ ] coverImage 文件名与标题对应
- [ ] categories 分类正确
- [ ] 文章 slug（URL）简洁且含核心关键词
```
