# Keganico SEO 检查清单

> 基于 keganico.com 现有文章 SEO 优化实践的检查清单

---

## 📋 发布前检查清单

### Title & Meta (必填)

```
□ Title 包含核心关键词
□ Title 长度 40-70 字符
□ Title 有吸引力 (问句/冒号/数字)
□ Meta Description 包含关键词
□ Meta Description 长度 150-160 字符
□ URL slug 简洁含关键词
□ slug 使用连字符分隔

示例检查:
✅ "14C28N Steel: Underrated or Overhyped?" (42 字符)
✅ "MagnaCut vs. S30V: Which Steel is Best for Your Next OEM Knife Project?" (72 字符)
❌ "Steel" (太短)
❌ "This is a very long title that exceeds the recommended character limit for SEO optimization" (太长)
```

### Content 内容

```
□ 开篇 100 词包含核心关键词
□ 开篇有**粗体核心定义/定位**
□ H2/H3 标题包含相关关键词
□ 关键词密度 1-2% (避免堆砌)
□ 文章长度达标:
  - 钢材评测：2500+ 词
  - 钢材对比：3000+ 词
  - 刀具知识：2000+ 词
  - 品牌评测：3500+ 词
  - 材料对比：2000+ 词
  - OEM 指南：3000+ 词
□ 段落简短 (2-4 句)
□ 使用列表增强可读性
□ 使用表格对比数据
```

### Links 内部链接

```
□ 内部链接总数 10-20 个
□ 开篇 2 段有 1-2 个链接
□ 每个 H2 章节至少 1 个链接
□ 锚文本描述性 (避免"点击这里")
□ 链接到相关产品页面
□ 链接到相关知识文章
□ 链接到服务页面 (OEM/联系)
□ 无死链 (所有链接有效)

链接分布检查:
□ 钢材页面：3-5 个
□ 对比文章：3-5 个
□ 产品页面：2-3 个
□ 知识页面：2-3 个
□ CTA 页面：2-3 个
```

### Images 图片

```
□ 每 400-500 字至少 1 张图片
□ 开篇后第 1 个章节有图
□ 每个 H2 章节有配图
□ 对比表格前后有图
□ 所有图片有 Alt 文本
□ Alt 文本 5-15 个单词
□ Alt 文本包含关键词
□ Alt 文本描述性 (避免"image of")
□ 图片文件名描述性
□ 图片文件名使用连字符
□ Figure 格式正确 (产品图)
□ 产品 Figure 有链接
□ 图片来源/数据有标注
□ 文件大小已优化 (<200KB)
```

### Structure 结构

```
□ H1 唯一 (文章标题)
□ H2/H3 层级清晰
□ H2 数量 5-10 个
□ H3 数量 5-15 个 (如需要)
□ 有目录/导航 (长文章)
□ 有总结/结论章节
□ 有 FAQ 章节 (3-8 个问题)
□ 有 CTA 章节
```

### CTA 行动号召

```
□ 文末有明确 CTA
□ CTA 有行动动词
□ CTA 链接有效
□ 联系方式清晰
□ OEM 服务提及
□ 批发选项提及
□ 价值主张明确
```

### B2B 元素 (Keganico 特色)

```
□ OEM 服务提及 (至少 1 次)
□ 批发选项提及
□ 联系/报价链接
□ 中国制造商视角
□ Yangjiang 产业优势 (如适用)
□ MOQ/价格区间 (如适用)
```

## 🔍 关键词优化检查

### 关键词布局

```
标题 (Title):
□ 核心关键词在前 50%
□ 长度适中 (40-70 字符)
□ 有吸引力 (问句/数字/冒号)

开篇段落:
□ 前 100 词出现核心关键词
□ 粗体强调核心定义
□ 自然融入，避免堆砌

H2/H3 标题:
□ 每 300-500 词包含相关关键词
□ 使用长尾变体
□ 描述性清晰

正文内容:
□ 关键词密度 1-2%
□ 自然分布，非堆砌
□ 使用同义词/变体

图片 Alt:
□ 描述性文字 + 关键词
□ 5-15 个单词
□ 避免"image of"开头

内部链接锚文本:
□ 使用关键词变体
□ 描述性清晰
□ 避免"点击这里"
```

### 关键词类型覆盖

```
□ 核心关键词 (1-2 个)
□ 长尾关键词 (3-5 个)
□ 对比关键词 (如适用)
□ 问题关键词 (FAQ 中)
□ 商业关键词 (CTA 中)
```

## 🖼️ 图片 SEO 检查

### Alt 文本规范

```
✅ 正确示例:
- "Performance scorecard comparison MagnaCut vs S30V steel"
- "14C28N steel chemical composition table"
- "Civivi Elementum folding knife with G10 handle"
- "Chef using knife for cutting vegetables"

❌ 错误示例:
- "image" (太短)
- "image of knife" (太简单)
- "DSC_2847" (无意义)
- "A picture showing a very sharp knife that is being used by a chef in a professional kitchen environment" (太长)
```

### 文件名规范

```
✅ 正确:
- performance-scorecard-magnacut-vs-s30v.jpg
- chemical-comparison-14c28n-vs-d2.jpg
- civivi-knives-overview.jpg

❌ 错误:
- DSC_2847.JPG
- 图片 1.jpg
- magnacut-vs-s30v-performance-scorecard-head-to-head-ratings-comparison-chart-final.jpg (过长)
```

### 图片质量

```
□ 清晰度足够 (最小 800px 宽)
□ 光线充足
□ 焦点清晰
□ 文件大小 <200KB
□ 格式正确 (JPG/PNG)
□ 无版权风险
```

## 🔗 内部链接检查

### 链接密度

```
□ 每 300-400 字至少 1 个链接
□ 开篇 2 段有 1-2 个链接
□ 每个 H2 章节至少 1 个链接
□ 整篇文章 10-20 个链接
□ 无链接堆砌 (单段不超过 2 个)
```

### 锚文本规范

```
✅ 正确:
- [OEM knife manufacturers](链接)
- [14C28N steel](链接)
- [G-10 handle material](链接)
- [EDC knife](链接)

❌ 错误:
- [点击这里](链接)
- [了解更多](链接)
- [这里](链接)
- [read more](链接)
```

### 链接目标多样性

```
□ 钢材页面 (3-5 个)
□ 对比文章 (3-5 个)
□ 产品页面 (2-3 个)
□ 知识页面 (2-3 个)
□ 服务页面 (2-3 个)
```

```

□ 图片 6-10 张

```
