# 发布前完整检查清单

**执行时机**: SEO 检查（`seo-checklist.md`）完成后，正式发布前
**用途**: 确保文章内容完整、图片真实、链接有效、用户体验良好

---

## ① 内容完整性

```markdown
- [ ] Front Matter 完整（title、date、categories、coverImage）
- [ ] 开篇 2-3 段，包含 1-2 个内部链接
- [ ] 每个主要章节有图片（至少每 500 字 1 张）
- [ ] 包含优缺点平衡分析
- [ ] 有对比章节（如适用）
- [ ] 结论段落包含 CTA 和 request-a-quote 链接
- [ ] 段落长度适中（3-5 句/段）
- [ ] 标题层级正确（## 主章节，### 子章节）
- [ ] 列表格式统一
- [ ] 无拼写和语法错误
- [ ] 使用第二人称 "you/your"
- [ ] 技术术语使用正确（压力等级、材质标准、连接方式）
```

---

## ② 图片资源验证 ⚠️

```markdown
- [ ] 所有图片链接来自网站现有资源或已下载到本地
- [ ] 未虚构不存在的图片文件名
- [ ] 图片年份优先使用 2026/2025 年资源
- [ ] 避免使用早于 2022 年的图片（除非必要）
- [ ] 图片描述与内容相关 ⭐ 重要
- [ ] 每章节至少有 1 张配图
- [ ] 图片文件名符合规范（小写 + 连字符）
- [ ] 图片已上传到正确路径（/wp-content/uploads/YYYY/MM/）
```

**图片相关性检查**（法兰/阀门/管件文章）:
```markdown
- [ ] 开篇图片：显示产品实物/应用场景，非随机工业图
- [ ] 规格章节：尺寸图/剖面图/技术图纸，非无关设备图
- [ ] 材质章节：材料样品/金相图/光谱分析图，非生锈金属图
- [ ] 压力等级：压力表/测试设备相关，非无关仪表图
- [ ] 安装章节：现场安装/工具使用图，非无关施工场景
- [ ] 应用章节：与描述行业匹配（化工厂/电厂/水处理）
- [ ] 对比章节：并排对比图/对比表格截图
- [ ] 避免使用无关的通用工业图片
```

---

## ③ 内部链接检查

```markdown
- [ ] 内部链接密度适当（每 300-400 字 1 个）
- [ ] 锚文本为描述性关键词，非"点击这里"
- [ ] 结尾包含 /request-a-quote/ 链接
- [ ] 所有内部链接 URL 有效，无 404
- [ ] 相关产品页面已链接
- [ ] 相关技术文档已链接
```

---

## ④ SEO 元素（快速核查）

```markdown
- [ ] Title 含核心关键词，长度 50-60 字符
- [ ] 开篇 100 词内出现核心关键词
- [ ] 至少 1 个 H2 含核心关键词变体
- [ ] 图片 Alt 属性描述性且含关键词
- [ ] 标准号正确引用（ASME B16.5, API 6D, ASTM A105）
```

> 完整 SEO 检查见 `qa/seo-checklist.md`

---

## ⑤ 技术准确性检查

```markdown
- [ ] 压力等级数据准确（Class 150 = 285 psi @ 100°F）
- [ ] 温度范围数据准确
- [ ] 材质化学成分正确
- [ ] 机械性能数值正确
- [ ] 标准号引用正确
- [ ] 尺寸规格符合行业标准
- [ ] 单位换算正确（psi/bar/MPa, °F/°C）
```

---

## ⑥ 用户体验

```markdown
- [ ] 文章结构清晰，长文章有目录导航
- [ ] 对比表格易于阅读，列宽合适
- [ ] CTA 位置明显且自然（不突兀）
- [ ] 段落不过长，移动端友好（每段不超过 5 句）
- [ ] 技术术语有适当解释
- [ ] 复杂概念有图表辅助说明
```

---

## ⑦ 发布信息确认

```markdown
- [ ] date 字段使用 2026 年
- [ ] coverImage 文件名与标题对应
- [ ] categories 分类正确（flanges/valves/pipe-fittings/industry-knowledge）
- [ ] 文章 slug（URL）简洁且含核心关键词
- [ ] Tags 已添加（3-5 个相关标签）
```

---

## ⑧ B2B 元素检查

```markdown
- [ ] 目标受众明确（承包商/采购商/工程师）
- [ ] 商业价值突出（成本效益/交货期/认证）
- [ ] OEM/ODM 服务能力已提及
- [ ] MOQ/批量价格信息已说明（如适用）
- [ ] 质检流程/认证已展示
- [ ] 项目案例/行业应用已列举
```
