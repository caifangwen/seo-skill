# SKILL-17: SEO 优化检查

**用途**: 发布前验证文章 SEO 质量
**执行时机**: 文章写作完成后，发布前

---

## SEO 核心元素检查

```markdown
- [ ] 核心关键词出现在 Title 中
- [ ] 核心关键词出现在开篇 100 词内
- [ ] 核心关键词出现在至少 1 个 H2 中
- [ ] 长尾关键词自然分布于全文
- [ ] 图片 Alt 属性包含描述性关键词
- [ ] 内部链接锚文本使用关键词变体
- [ ] Title 长度 50-60 字符
- [ ] Meta Description 长度 150-160 字符（如单独设置）
```

---

## 关键词布局验证

```markdown
- [ ] 已确认目标关键词搜索意图匹配文章内容
- [ ] 已验证关键词难度（KD）适合当前域名权重
- [ ] 已规划关键词布局位置（Title / H2 / 开篇 / Alt）
- [ ] B2B 关键词已适当融入（wholesale, supplier, manufacturer）
- [ ] 对比关键词已覆盖（如适用：A vs B, X vs Y）
- [ ] 问题关键词已在 FAQ 中回答（如适用）
- [ ] 标准号关键词已覆盖（ASME, API, ASTM, DIN）
```

---

## 竞对分析验证

```markdown
- [ ] 已分析前 3 名竞争对手内容
- [ ] 已识别内容差距（Content Gap）
- [ ] 已添加竞争对手缺失的主题
- [ ] 已强化差异化优势（B2B 视角、技术数据、标准合规）
- [ ] 已规划反向链接获取策略
```

---

## 结构化数据机会

- **FAQ 章节** → 可触发 Google 精选摘要（Featured Snippet）
- **对比表格** → 可触发表格摘要
- **How-to 步骤** → 可触发步骤摘要
- **产品规格表** → 可触发数据摘要

确保 FAQ、对比表格、步骤类内容格式规范，使用标准 Markdown 语法。

---

## 行业特定 SEO 检查

```markdown
- [ ] 产品材质标准已标注（ASTM A105, A182 F304, etc.）
- [ ] 压力等级已明确（Class 150, PN16, etc.）
- [ ] 尺寸范围已说明（NPS 1/2" to 24"）
- [ ] 应用行业已列出（Oil & Gas, Chemical, Water Treatment）
- [ ] 认证信息已提及（API 6D, PED, ISO 9001）
```
