# Keganico 发布前检查清单

> 确保每篇文章发布前都经过完整的质量检查

---

## 📋 内容质量检查

### 信息准确性

```
□ 所有数据有来源标注
□ 化学成分数据准确
□ 性能数据有引用
□ 价格信息最新
□ 品牌/公司信息准确
□ 年份信息更新 (2026)
□ 链接目标页面有效
```

### 逻辑清晰度

```
□ 文章结构合理
□ 章节过渡自然
□ 论点有数据支撑
□ 对比公平客观
□ 结论有依据
□ 无自相矛盾
```

### 语言流畅性

```
□ 无拼写错误
□ 无语法错误
□ 句子通顺
□ 段落连贯
□ 术语一致
□ 语气统一 (专业友好)
```

### 价值明确性

```
□ 开篇明确文章价值
□ 每章有明确目的
□ 结论有 actionable 建议
□ B2B 读者有收获
□ CTA 明确有价值
```

---

## 📐 格式规范检查

### Front Matter

```
□ title 字段存在
□ date 字段存在 (YYYY-MM-DD)
□ categories 字段存在
□ coverImage 字段存在
□ 分类选择正确
□ 日期为当前/未来日期
```

### 标题层级

```
□ H1 唯一 (文章标题)
□ H2 数量 5-10 个
□ H3 使用正确 (如需要)
□ 无跳过层级
□ 标题描述清晰
```

### 段落格式

```
□ 段落简短 (2-4 句)
□ 无大段文字块
□ 列表使用正确
□ 表格格式正确
□ 粗体使用恰当
□ 斜体使用恰当
```

### 代码/引用格式

```
□ 引用有标注
□ 数据有来源
□ Figure 格式正确
□ 表格有标题/说明
```

## 🔗 链接检查

### 内部链接

```
□ 数量 10-20 个
□ 所有链接有效
□ 锚文本描述性
□ 分布均匀
□ 包含相关产品
□ 包含相关知识
□ 包含服务页面
```

### 外部链接

```
□ 来源链接有效
□ 权威来源优先
□ 数量适中 (1-5 个)
□ 无竞争网站链接
□ 新标签打开 (如需要)
```

### CTA 链接

```
□ OEM 服务链接
□ 联系/报价链接
□ 产品类别链接
□ 所有 CTA 链接有效
```

---

## 🖼️ 图片检查

### 数量检查

```
□ 每 400-500 字 1 张图
□ 总数符合文章类型要求:
  - 钢材评测：10-14 张
  - 钢材对比：7-10 张
  - 刀具知识：9-13 张
  - 品牌评测：12-18 张
  - 材料对比：6-10 张
  - OEM 指南：8-12 张
```

### 质量检查

```
□ 图片清晰
□ 尺寸合适 (最小 800px 宽)
□ 文件大小优化 (<200KB)
□ 格式正确 (JPG/PNG)
□ 无版权风险
```

### 规范检查

```
□ 所有图片有 Alt 文本
□ Alt 文本 5-15 词
□ Alt 文本描述性
□ 文件名描述性
□ 文件名使用连字符
□ Figure 格式正确 (产品图)
□ 产品 Figure 有链接
□ 数据来源有标注
```

### 位置检查

```
□ 开篇后第 1 章有图
□ 每个 H2 章节有图
□ 对比表格前后有图
□ 分布均匀
```

## 🎯 SEO 检查

```

□ Alt 文本含关键词
□ 锚文本含关键词
□ 关键词密度 1-2%
```

### Meta 信息

```
□ Title 40-70 字符
□ Meta Description 150-160 字符
□ URL slug 简洁
□ slug 含关键词
```

### 技术 SEO

```
□ 无重复内容
□ 无死链
□ 图片有 Alt
□ 结构清晰
```

## 📊 检查评分

### 优秀 (95-100 分)

```
□ 所有检查项通过
□ 无任何问题
□ 可直接发布
```

### 良好 (85-94 分)

```
□ 大部分检查项通过
□ 少量小问题
□ 修复后可发布
```

### 需改进 (70-84 分)

```
□ 部分检查项未通过
□ 需要中等修复
□ 修复后重新检查
```

### 不合格 (<70 分)

```
□ 多项检查项未通过
□ 需要大量修复
□ 不建议发布
```

## 🔄 检查流程

### 自检流程 (作者)

```
1. 完成初稿
2. 内容质量检查
3. 格式规范检查
4. 链接图片检查
5. SEO 基础检查
6. 提交审核
```

### 审核流程 (编辑)

```
1. 内容质量复审
2. 格式规范复审
3. 链接图片抽检
4. SEO 深度检查
5. B2B 元素检查
6. 批准/修改意见
```

### 发布流程 (运营)

```
1. 最终格式检查
2. 发布前预览
3. 定时发布
4. 发布后检查
5. 推广执行
```
