# SKILL-17: SEO 优化检查

**用途**: 发布前检查 SEO 元素

---

## SEO 检查清单

### Title & Meta

- [ ] Title 包含核心关键词 (40-70 字符)
- [ ] 描述包含关键词 (150-160 字符)
- [ ] URL slug 简洁含关键词

### Content

- [ ] 开篇 100 词包含核心关键词
- [ ] H2/H3 标题包含相关关键词
- [ ] 关键词密度 1-2% (避免堆砌)
- [ ] 文章长度 2000+ 词

### Links

- [ ] 内部链接 10-20 个
- [ ] 外部权威链接 1-3 个 (可选)
- [ ] 锚文本描述性 (避免"点击这里")

### Images

- [ ] 每 400-500 字至少 1 张图片
- [ ] 所有图片有 Alt 文本
- [ ] Alt 文本包含关键词
- [ ] 图片文件名描述性

### Structure

- [ ] H1 唯一 (文章标题)
- [ ] H2/H3 层级清晰
- [ ] 段落简短 (2-4 句)
- [ ] 使用列表/表格增强可读性

### CTA

- [ ] 文末有明确 CTA
- [ ] CTA 链接有效
- [ ] 联系方式清晰

### B2B Elements

- [ ] OEM 服务提及
- [ ] 批发选项
- [ ] 联系/报价链接
