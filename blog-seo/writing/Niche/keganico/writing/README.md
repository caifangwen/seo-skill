## 🔄 完整工作流程

```
写作前准备 (30-60min):
├─ 00-research: 关键词调研
└─ 00-research: 竞对分析

写作阶段 (90-150min):
├─ 01-front-matter
├─ 02-intro
├─ [根据文章类型选择技能]
│   ├─ 钢材类：03, 04, 14
│   ├─ 对比类：05, 14, 15
│   ├─ 知识类：09, 10, 12, 13
│   └─ 评测类：11, 12, 15
└─ 08-cta + 16-verdict

优化阶段 (20-30min):
├─ 07-internal-links: 内链植入
├─ 06-images: 图片配置
└─ 17-seo-checklist: SEO 检查

发布前:
└─ 18-content-gap + 检查清单验证
```
