# SEO 文章写作技能

## 概述

本技能帮助你创建高质量、符合 SEO 最佳实践的文章。它包括：
- 关键词研究和优化
- 内容结构优化（标题层级、段落结构）
- 元数据生成（标题、描述）
- 自动下载相关图片并正确引用

## 快速开始

当用户请求创建 SEO 文章时，按照以下流程工作：

1. **确定主题和关键词** - 与用户确认目标关键词和主题
2. **研究竞争对手** - 分析排名靠前的内容
3. **创建大纲** - 设计符合 SEO 的标题结构
4. **撰写内容** - 遵循 SEO 写作最佳实践
5. **下载图片** - 使用脚本下载相关图片
6. **引用图片** - 在文章中正确插入图片
7. **生成元数据** - 创建 SEO 标题和描述

## SEO 写作指南

### 关键词优化

- **主关键词密度**: 1-2%（每 1000 字出现 10-20 次）
- **关键词位置**:
  - 标题开头
  - 第一段（前 100 字）
  - 至少一个 H2 标题
  - 结论段落
  - 图片 alt 文本
- **长尾关键词**: 使用 3-5 个相关长尾变体
- **LSI 关键词**: 包含语义相关的术语

### 标题结构

```
H1: 主标题（包含主关键词，50-60 字符）
├── H2: 主要章节（包含关键词变体）
│   ├── H3: 子章节（可选）
│   └── H3: 子章节（可选）
├── H2: 主要章节
└── H2: 结论/总结
```

**规则**:
- 每篇文章只有一个 H1
- H2 之间逻辑连贯
- 标题长度：H1 (50-60 字), H2 (20-40 字)

### 内容优化

- **引言**: 100-150 字，包含主关键词，吸引读者
- **段落长度**: 2-4 句，避免大段文字
- **列表使用**: 每 300 字至少一个列表（有序或无序）
- **内部链接**: 3-5 个相关内部链接
- **外部链接**: 2-3 个权威来源
- **字数**: 最少 1000 字，目标 1500-2500 字

### 元数据

```markdown
SEO 标题：50-60 字符，包含主关键词
Meta 描述：150-160 字符，包含关键词和行动号召
URL Slug: 简短，包含关键词，用小写字母和连字符
```

## 图片处理

### 下载图片

使用 `download_images.py` 脚本下载与文章主题相关的图片：

```bash
python scripts/download_images.py --query "关键词" --output images/ --count 5
```

**参数**:
- `--query`: 搜索关键词（必需）
- `--output`: 输出目录（默认：`images/`）
- `--count`: 下载数量（默认：5，最大：20）
- `--size`: 图片尺寸 `small`/`medium`/`large`（默认：`medium`）

### 图片引用规范

在 Markdown 文章中引用图片：

```markdown
![描述文字，包含关键词](images/image-name.webp)
*图片说明：图片来源或补充说明*
```

**规则**:
- Alt 文本必须描述图片内容并包含关键词
- 图片文件名使用小写，连字符分隔
- 使用 WebP 格式（更好的压缩，更小的文件大小）
- 每 300-500 字插入一张图片
- 首段后必须有至少一张图片

### 图片优化检查清单

- [ ] 文件名描述性（包含关键词）
- [ ] Alt 文本准确描述
- [ ] 图片大小适中（<200KB）
- [ ] 格式正确（JPG 用于照片，PNG 用于图表）
- [ ] 有图片来源说明（如需要）

## 脚本使用

### download_images.py

下载与主题相关的免版税图片。

```bash
# 基本用法
python scripts/download_images.py --query "人工智能技术"

# 指定输出目录和数量
python scripts/download_images.py --query "机器学习" --output article-images/ --count 10

# 指定图片尺寸
python scripts/download_images.py --query "数据科学" --size large
```

**依赖**:
- `requests`
- `Pillow`

安装依赖：
```bash
pip install requests Pillow
```

### update_article_images.py

自动替换文章中的占位符图片（如 `featured-image.jpg`, `image-1.jpg` 等）为实际下载的图片。

```bash
# 自动替换占位符并插入新图片
python scripts/update_article_images.py --article article.md --images-dir images/

# 仅替换占位符
python scripts/update_article_images.py --article article.md --images-dir images/ --replace-only

# 列出图片和引用示例
python scripts/update_article_images.py --list images/ --keyword "人工智能"
```

**功能**:
- 自动识别占位符图片：`featured-image.jpg`, `image-1.jpg`, `image-2.jpg` 等
- 根据章节上下文生成准确的 alt 文本
- 添加图片尺寸和文件大小信息
