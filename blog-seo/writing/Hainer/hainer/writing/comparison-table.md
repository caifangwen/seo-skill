# SKILL-05: 产品对比表格

**用途**: 生成对比表格
**适用**: 产品对比文章、选型指南文章、排行榜文章

---

## 结构模板

```markdown
## Comparison Table: [产品 A] vs. [产品 B]

![对比图片](https://img.example.com/wp-content/uploads/YYYY/MM/图片名.jpg)

In the comparison table below, you'll find the features, pros, and cons of [A] and [B].

|  | [产品 A] | [产品 B] |
| --- | --- | --- |
| Features | 特点 1<br>特点 2 | 特点 1<br>特点 2 |
| Pros | 优点 1<br>优点 2 | 优点 1<br>优点 2 |
| Cons | 缺点 1<br>缺点 2 | 缺点 1<br>缺点 2 |
```

---

## 法兰对比示例（Class 150 vs Class 300）

```markdown
## Comparison: Class 150 vs Class 300 Flanges

|  | **Class 150** | **Class 300** |
| --- | --- | --- |
| **Pressure Rating** | 285 psi @ 100°F | 740 psi @ 100°F |
| **Wall Thickness** | Standard | Heavier |
| **Weight** | Lighter | ~40% heavier |
| **Cost** | Lower | Higher |
| **Applications** | Low pressure, water, air | Medium pressure, oil, gas |
| **Bolt Size** | Smaller | Larger |
```

---

## 阀门对比示例（Ball Valve vs Gate Valve）

```markdown
## Comparison: Ball Valve vs Gate Valve

|  | **Ball Valve** | **Gate Valve** |
| --- | --- | --- |
| **Operation** | Quarter-turn (90°) | Multi-turn |
| **Flow Control** | On/Off only | On/Off only |
| **Pressure Drop** | Low | Very low (when fully open) |
| **Sealing** | Bubble-tight | Good, but may leak over time |
| **Speed** | Fast operation | Slower operation |
| **Cost** | Moderate | Lower |
| **Maintenance** | Low | Moderate |
```

## 材料对比示例（Carbon Steel vs Stainless Steel）

```markdown
## Comparison: Carbon Steel vs Stainless Steel Flanges

|  | **Carbon Steel (A105)** | **Stainless Steel (F304)** |
| --- | --- | --- |
| **Corrosion Resistance** | Low (requires coating) | Excellent |
| **Temperature Range** | -20°F to 800°F | -425°F to 1550°F |
| **Cost** | Lower | 2-3x higher |
| **Applications** | General purpose | Corrosive media, food grade |
| **Maintenance** | Regular painting required | Minimal |
| **Weldability** | Good | Good (requires proper technique) |
```
