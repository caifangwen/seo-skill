## 示例（法兰安装）

```markdown
## How to Install Weld Neck Flanges?

Proper flange installation ensures leak-free operation and system integrity. Here are the step-by-step instructions for welding neck flanges:

### Preparation

Before installation, inspect all components and prepare the work area:

- Check flange material certificates match project specifications
- Verify flange rating and size match the piping requirements
- Inspect flange faces for damage or machining defects
- Clean pipe ends and remove any rust, oil, or debris

### Tools Required

- Torque wrench
- Dial indicator for alignment
- Welding equipment (SMAW/GTAW)
- Gasket
- Bolts and nuts (properly lubricated)
- Spirit level

### Installation Steps

1. **Fit-up and Alignment**: Slide the flange onto the pipe and ensure proper fit-up. The flange hub should sit flush against the pipe end with correct root gap for welding.

2. **Tack Welding**: Make 3-4 tack welds evenly spaced around the circumference to hold the flange in position. Check flange perpendicularity using a spirit level.

3. **Root Pass Welding**: Complete the root pass weld using appropriate filler material. For stainless steel, use back purging with argon to prevent oxidation.

4. **Fill and Cap Passes**: Complete the remaining weld passes according to WPS (Welding Procedure Specification). Maintain interpass temperature within specified limits.

5. **Bolt Installation**: After welding and cooling, place the gasket centered on the flange face. Insert bolts and hand-tighten all nuts.

6. **Torque Tightening**: Using a criss-cross pattern, tighten bolts to 30%, 60%, and finally 100% of specified torque in three stages.

### Post-Installation Checks

- Visually inspect all welds for defects
- Perform NDT testing if required by project specifications
- Conduct hydrostatic pressure testing
- Check for leaks using soap solution or leak detection fluid
- Mark completed joints with identification tags
```

## 示例（阀门安装）

```markdown
## How to Install a Ball Valve?

Correct valve installation ensures optimal performance and extended service life. Follow these steps for proper ball valve installation:

### Preparation

- Verify valve pressure rating matches system requirements
- Check valve material compatibility with the fluid being handled
- Inspect valve ends and sealing surfaces for damage
- Ensure the valve is in the closed position during installation

### Installation Steps

1. **Position the Valve**: Place the valve between pipe flanges or prepare for welding. Ensure the flow direction arrow on the valve body matches the intended flow direction.

2. **Flange Alignment**: For flanged valves, insert gaskets on both sides. Align bolt holes and insert bolts hand-tight.

3. **Bolt Tightening**: Tighten flange bolts in a criss-cross pattern to ensure even gasket compression. Follow the manufacturer's torque specifications.

4. **Welded Connection**: For butt-weld ends, ensure proper bevel angle (typically 37.5°) and root gap (3-4mm). Weld according to qualified procedure.

5. **Operational Check**: After installation, cycle the valve from fully open to fully closed to ensure smooth operation. Verify the handle or actuator has full range of motion.

### Important Notes

- Do not use the valve handle for lifting or supporting the valve
- For buried service, apply appropriate coating and cathodic protection
- Install valves in accessible locations for future maintenance
- Consider installing isolation valves upstream and downstream for easy maintenance
```
