# 写作风格规范

**用途**: 所有法兰/阀门/管件英文博客文章的语言风格标准
**适用**: 写作全程参考

---

## 人称使用

| 人称 | 用途 |
|-----|------|
| **第二人称** "you/your" | 主要人称，与读者对话 |
| **第一人称** "we/us" | 代表公司发言时 |
| **第三人称** | 客观描述事实时 |

---

## 语气特点

- **专业但平易近人** — 避免过于学术化，也不要过度口语化
- **引导式** — 多用问句带入话题（"Wondering which flange class to choose?"）
- **数据支撑** — 提供具体数值（压力等级、温度范围、材质成分百分比等）
- **平衡客观** — 同时说明优缺点，不片面吹捧

---

## 段落规范

- 每段 **3-5 句话**
- 每句 **15-25 个单词**
- 避免超过 6 行的段落
- 移动端友好：段落之间留空行

---

## 标题规范

- `##` 主章节标题（每篇文章 5-10 个）
- `###` 子章节标题
- 标题首字母大写（Title Case）
- 标题中自然融入关键词变体，不强行堆砌

---

## 常见错误避免

```
❌ 避免："Click here to learn more"
✅ 改为：使用描述性锚文本链接

❌ 避免：段落过长（10+ 句连续）
✅ 改为：每 3-5 句换段

❌ 避免：全大写强调 "THIS IS VERY IMPORTANT"
✅ 改为：用 **加粗** 或结构化列表强调

❌ 避免：只写优点，不提缺点
✅ 改为：始终保持 Pros/Cons 平衡

❌ 避免：虚构图片文件名
✅ 改为：只引用确认存在的图片资源
```

## B2B 写作视角

本网站的核心受众是**管道工程承包商、工业设备采购商和分销商**，而非终端消费者。
写作时始终保持这一视角：

- 提及"your clients/projects"而非"you（使用者）"
- 强调商业价值：项目合规性、交付可靠性、采购成本效益
- 适时引入批发/OEM 服务 CTA
- 举例时用"contractors"、"procurement managers"或"distributors"作为受众对象
- 强调行业标准合规性（ASME、API、ANSI、DIN 等）

## 行业术语规范

| 类别 | 正确示例 |
|-----|---------|
| **压力等级** | Class 150, Class 300, PN10, PN16 |
| **材质标准** | ASTM A105, A182 F304, A234 WPB |
| **尺寸标准** | ASME B16.5, ASME B16.9, DIN 2501 |
| **连接方式** | Weld Neck, Socket Weld, Threaded, Flanged |
| **应用场景** | Oil & Gas, Petrochemical, Water Treatment, Power Generation |
