## 📝 类型 1: 钢材评测 (Steel Review)

### 适用主题

```
- [钢材名] Steel Review
- Is [钢材名] Steel Good?
- [钢材名] Steel: Complete Guide
- Everything About [钢材名] Steel
```

### 标准结构

```markdown
1. Front Matter
2. 开篇引入 (钢材定位 + 核心特点)
   - 常见误解澄清
   - 实际定位说明
   - 内容预告

3. What is [钢材名] Steel?
   - 定义/开发商
   - 历史背景
   - 市场定位

4. Chemical Composition
   - 成分表格 (Figure 格式)
   - 关键元素解析
   - 数据来源

5. Performance Analysis
   - Performance Overview Table (可选)
   - Toughness (韧性)
   - Corrosion Resistance (耐腐蚀)
   - Edge Retention (边缘保持)
   - Ease of Sharpening (易磨性)
   - Hardness Range (硬度范围)

6. [钢材名] vs Other Steels
   - 2-4 个对比表格
   - 详细分析
   - 内部链接

7. Best Uses & Limitations
   - 最佳用途 (3-5 个)
   - 限制/不推荐用途

8. FAQ (4-6 个问题)

9. CTA (OEM/批发导向)
```

### 图片配置

```
封面图 × 1
成分表配图 × 1
性能章节图 × 4-5 (每节 1 张)
对比章节图 × 2-3
产品展示图 × 2-3 (Figure 格式)
总计：10-14 张
```

### 内链策略

```
钢材页面：3-5 个
对比文章：3-5 个
产品页面：2-3 个
知识页面：2-3 个
CTA: 2-3 个
总计：12-19 个
```

### 创作流程

```
1. 关键词研究 (20min)
   - 核心关键词
   - 长尾词 5-10 个
   - 竞品分析

2. 数据收集 (30min)
   - 化学成分
   - 性能数据 (Knife Steel Nerds 等)
   - 硬度 HRC 范围

3. 大纲制定 (15min)
   - H2/H3 结构
   - 图片位置
   - 内链规划

4. 内容撰写 (60-90min)
   - 开篇
   - 主体章节
   - FAQ
   - CTA

5. 优化审核 (30min)
   - 内链植入
   - 图片配置
   - SEO 检查

总计：2.5-3.5 小时
```

### 示例文章

```
- 14C28N Steel: Underrated or Overhyped?
- Nitro-V Steel: Complete Guide
- CruWear Steel Review
- MagnaCut Steel Review
```

## 📝 类型 2: 钢材对比 (Steel Comparison)

### 适用主题

```
- [A] vs [B]
- [A] vs [B] vs [C]
- [钢材] Compared: Which is Better?
```

### 标准结构

```markdown
1. Front Matter
2. 开篇引入
   - 对比背景
   - 核心差异概述
   - B2B 视角 (如适用)

3. Steel Overview
   - [A] 概述 (历史/定位)
   - [B] 概述 (历史/定位)

4. Chemical Composition Comparison
   - 对比表格
   - 详细解读
   - 数据来源

5. Performance Scorecard
   - 评分对比表
   - 每项详细分析
     - Edge Retention
     - Toughness
     - Corrosion Resistance
     - Ease of Sharpening
     - Hardness

6. Application Guide
   - Scenario A: 推荐 + 商业案例
   - Scenario B: 推荐 + 商业案例
   - Scenario C: 推荐 + 商业案例

7. Cost Analysis & Market Pricing
   - 价格对比表
   - 市场策略建议

8. CTA (OEM 合作导向)

9. Related Reading (3-6 个相关链接)
```

### 图片配置

```
封面图 × 1
成分对比图 × 1
性能评分图 × 1
应用场景图 × 2-3
产品展示图 × 2-3
总计：7-10 张
```

### 创作流程

```
1. 数据收集 (40min)
   - 双方化学成分
   - 性能测试数据
   - 价格信息

2. 对比测试分析 (30min)
   - 同条件对比
   - 数据记录
   - 结论提炼

3. 内容撰写 (90-120min)
   - 开篇对比
   - 详细分析
   - 应用建议
   - 价格分析

4. 图片处理 (30min)
   - 对比图制作
   - 产品图配置

5. 审核发布 (30min)

总计：3.5-4.5 小时
```

### 示例文章

```
- MagnaCut vs. S30V: Which Steel is Best?
- S35VN vs D2 Steel Comparison
- 14C28N vs D2: Ultimate Comparison
- Nitro-V vs 14C28N
```
