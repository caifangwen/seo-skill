# 文章类型组合流程

根据文章类型选择对应的技能模块组合。

---

## 类型 1: 钢材评测文章

**技能链**: `keyword-research` → `competitor-research` → `frontmatter` → `intro` → `steel-specs` → `pros-cons` → `use-cases` → `conclusion-cta`

```
关键词调研 → 竞对分析 → Front Matter → 开篇引言
→ 成分列表 + 属性详解 → 优缺点 → 使用场景 → 结论 CTA
```

---

## 类型 2: 钢材对比文章

```

→ 分别介绍 → 对比表格 → 属性对比 → 购买建议
```

## 类型 7: 排行榜 / 最佳推荐文章

**技能链**: `keyword-research` → `competitor-research` → `frontmatter` → `intro` → `pros-cons` → `comparison-table` → `conclusion-cta`

```
关键词调研 → 竞对分析 → Front Matter → 开篇引言
→ 分类推荐 → 详细评测 → 对比表格 → 购买建议
```

**示例结构** (Best Pocket Knife Steels):
```markdown
1. 开篇：引入主题 + 核心关键词
2. 钢材知识科普（Understanding knife steel）
3. 评判标准（Factors determining quality）
4. 分级推荐（Low-end → Mid-range → High-end）
5. 综合对比表格
6. 购买建议 + CTA
```
