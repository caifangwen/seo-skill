# 文章类型组合流程

根据文章类型选择对应的技能模块组合。

---

## 类型 1: 产品详解文章

**技能链**: `keyword-research` → `competitor-research` → `frontmatter` → `intro` → `material-specs` → `pros-cons` → `use-cases` → `installation` → `conclusion-cta`

```
关键词调研 → 竞对分析 → Front Matter → 开篇引言
→ 材料规格 + 属性详解 → 优缺点 → 使用场景 → 安装指南 → 结论 CTA
```

**适用**: 法兰详解、阀门详解、管件详解

---

## 类型 2: 产品对比文章

**技能链**: `keyword-research` → `competitor-research` → `frontmatter` → `intro` → `comparison-table` → `material-specs` → `material-comparison` → `conclusion-cta`

```
关键词调研 → 竞对分析 → Front Matter → 开篇引言
→ 分别介绍 → 对比表格 → 材料/性能对比 → 选型建议
```

**示例**: Class 150 vs Class 300 Flanges, Ball Valve vs Gate Valve, 304 vs 316 Stainless Steel

---

## 类型 3: 安装/选型指南文章

**技能链**: `keyword-research` → `competitor-research` → `frontmatter` → `intro` → `material-specs` → `installation` → `faq` → `conclusion-cta`

```
关键词调研 → 竞对分析 → Front Matter → 开篇引言
→ 产品规格 → 安装步骤 → 注意事项 → FAQ → 结论 CTA
```

**适用**: 法兰安装指南、阀门选型指南、管道连接教程

---

## 类型 4: 材料介绍文章

**技能链**: `keyword-research` → `competitor-research` → `frontmatter` → `intro` → `material-specs` → `material-comparison` → `pros-cons` → `use-cases` → `conclusion-cta`

```
关键词调研 → 竞对分析 → Front Matter → 开篇引言
→ 材料定义/成分 → 对比分析 → 优缺点 → 使用场景 → 结论 CTA
```

**适用**: ASTM A105 详解、不锈钢材质介绍、合金钢应用

---

## 类型 5: 价格分析文章

**技能链**: `keyword-research` → `competitor-research` → `frontmatter` → `intro` → `price-analysis` → `material-comparison` → `conclusion-cta`

```
关键词调研 → 竞对分析 → Front Matter → 开篇引言
→ 价格因素 → 成本对比 → 投资建议 → 结论 CTA
```

**适用**: 法兰价格分析、阀门成本对比、材质选择性价比

---

## 类型 6: 知识科普文章

**技能链**: `keyword-research` → `competitor-research` → `frontmatter` → `intro` → `history` → `material-specs` → `use-cases` → `faq` → `conclusion-cta`

```
关键词调研 → 竞对分析 → Front Matter → 开篇引言
→ 历史背景 → 分类/标准介绍 → 使用场景 → FAQ → 结论 CTA
```

**适用**: 法兰标准解读、阀门类型介绍、管道基础知识

---

## 类型 7: 行业应用文章

**技能链**: `keyword-research` → `competitor-research` → `frontmatter` → `intro` → `use-cases` → `material-specs` → `pros-cons` → `conclusion-cta`

```
关键词调研 → 竞对分析 → Front Matter → 开篇引言
→ 行业需求分析 → 适用产品 → 优缺点 → 应用案例 → 结论 CTA
```

**适用**: 石化行业应用、水处理系统、电力行业管道

---

## 类型 8: 标准解读文章

**技能链**: `keyword-research` → `competitor-research` → `frontmatter` → `intro` → `material-specs` → `comparison-table` → `faq` → `conclusion-cta`

```
关键词调研 → 竞对分析 → Front Matter → 开篇引言
→ 标准概述 → 关键要求 → 对比分析 → FAQ → 结论 CTA
```

**适用**: ASME B16.5 解读、API 6D 要求、ASTM 材质标准

## 文章结构示例

### 示例 1: 法兰对比文章 (Class 150 vs Class 300)

```markdown
1. 开篇：引入压力等级概念 + 核心关键词
2. 法兰分类概述 (Class 150, Class 300, Class 600)
3. 压力 - 温度额定值对比
4. 尺寸和重量差异
5. 材质选择建议
6. 应用场景对比
7. 安装注意事项
8. 综合对比表格
9. 选型建议 + CTA
```

### 示例 2: 阀门选型指南

```markdown
1. 开篇：阀门在管道系统中的重要性
2. 主要阀门类型介绍 (Ball, Gate, Globe, Butterfly, Check)
3. 选型关键因素 (压力、温度、介质、流量)
4. 各类型阀门详细对比
5. 行业应用推荐
6. 执行器选型简介
7. 安装和维护要点
8. FAQ
9. 结论 + CTA
```

### 示例 3: 材质介绍文章 (Stainless Steel 316)

```markdown
1. 开篇：316 不锈钢的行业地位
2. 化学成分和机械性能
3. 与 304 不锈钢的对比
4. 耐腐蚀性能详解
5. 温度范围和应用
6. 焊接和加工指南
7. 价格因素分析
8. 典型应用案例
9. 结论 + CTA
```
