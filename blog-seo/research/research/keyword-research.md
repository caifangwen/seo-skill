## 1. 关键词类型分类

| 类型 | 说明 | 示例 |
|-----|------|------|
| 核心关键词 | 文章主要目标词 | "flange types", "ball valve guide" |
| 长尾关键词 | 3-5 词的具体查询 | "class 150 vs 300 flange pressure rating" |
| 对比关键词 | 包含对比意图 | "ball valve vs gate valve", "304 vs 316 flange" |
| 问题关键词 | 疑问句形式 | "what is a weld neck flange", "how to install flange" |
| 商业关键词 | 含购买意图 | "buy industrial flanges", "wholesale pipe fittings" |

## 2. 关键词调研步骤

**步骤 1: 确定主题范围**
```
输入：产品类型/主题
输出：核心关键词列表

示例 - 法兰主题:
- [产品类型] flanges
- [材质] flange specifications
- [类型 A] vs [类型 B] flange
- what is [法兰类型]
- [法兰] pressure rating chart

示例 - 阀门主题:
- [阀门类型] valve
- how does a [阀门] work
- [阀门 A] vs [阀门 B]
- best valve for [应用]
- [阀门] size chart
```

**步骤 2: 分析搜索意图**
```
信息型 (Informational):
- "what is...", "how to...", "guide to..."
- 适合：科普文章、安装指南、选型指南

商业型 (Commercial):
- "best...", "top...", "comparison..."
- 适合：产品对比、评测文章、排行榜

交易型 (Transactional):
- "buy...", "wholesale...", "supplier...", "manufacturer..."
- 适合：产品页面、B2B 页面、询价页面
```

**步骤 3: 关键词难度评估**
```
低难度 (KD 0-30):
- 长尾词、细分主题
- 示例："duplex 2205 flange properties", "API 6D valve requirements"

中难度 (KD 30-60):
- 中等竞争词
- 示例："flange pressure classes", "ball valve types", "stainless steel fittings"

高难度 (KD 60+):
- 核心大词
- 示例："pipe fittings", "industrial valves", "steel flanges"
```

## 4. 关键词布局规则

**标题 (Title)**: 包含核心关键词，长度 50-60 字符
```
示例："Class 150 vs Class 300 Flanges: Key Differences Explained"
示例："Ball Valve Types: A Complete Guide for Industrial Applications"
```

**开篇段落**: 前 100 词出现核心关键词，自然融入，避免堆砌

**副标题 (H2/H3)**: 每 300-500 词包含 1 个相关关键词，使用长尾变体
```
示例："## Flange Pressure Ratings", "## Stainless Steel vs Carbon Steel"
示例："## Installation Steps", "## Common Applications"
```

**图片 Alt**: 描述性文字 + 关键词
```
示例：alt="weld neck flange dimensions ASME B16.5"
示例：alt="ball valve cross section diagram"
```

**内部链接锚文本**: 使用关键词变体
```
示例：[flange pressure class chart](https://example.com/flange-pressure-chart/)
示例：[stainless steel valve suppliers](https://example.com/stainless-steel-valves/)
```
