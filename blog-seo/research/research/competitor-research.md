# SKILL-00B + SKILL-18: 竞对调研与内容差距分析

**用途**: 分析竞争对手内容，找出差距和机会
**执行时机**: 关键词调研（`keyword-research.md`）完成后

---

## 1. 竞对识别

**直接竞争对手**:
- 法兰/阀门制造商网站
- 工业管道产品供应商博客
- 行业权威站点（如 engineeringtoolbox.com, pipefittingworld.com）

**间接竞争对手**:
- 工程咨询公司技术文档
- 行业协会资源页（如 ASME, API）
- YouTube 技术频道（安装视频、产品评测）
- 行业论坛（如 Eng-Tips, CR4）

---

## 2. 竞对分析维度

**内容结构分析**:
```
□ 文章长度（字数）
□ 章节数量
□ 图片/图表数量
□ 技术数据表使用情况
□ 视频嵌入
□ FAQ 章节
□ CTA 位置和形式
```

**SEO 元素分析**:
```
□ Title 格式和长度
□ Meta Description
□ H1/H2/H3 结构
□ 内部链接数量
□ 外部链接来源（标准组织、权威站点）
□ 关键词密度
□ Schema 标记使用
```

**技术深度分析**:
```
□ 是否提供详细规格数据
□ 是否引用行业标准（ASME/API/ASTM）
□ 是否包含计算工具/图表
□ 是否提供 CAD 图纸下载
□ 是否包含材料证书样本
```

**用户体验分析**:
```
□ 页面加载速度
□ 移动端适配
□ 技术文档可读性
□ 导航清晰度
□ 询盘/联系入口
```

## 3. 竞对分析模板

```markdown
## 竞争对手分析报告

**目标关键词**: ball valve vs gate valve

### 竞争对手 1: [网站名]
URL: [链接]

**内容分析**:
- 字数：约 2,500 词
- 章节：6 个 H2, 10 个 H3
- 图片：8 张（产品图 + 原理图）
- 表格：3 个对比表
- CTA: 文末 1 处 + 侧边栏

**SEO 分析**:
- Title: "Ball Valve vs Gate Valve: What's the Difference?"（52 字符）
- H1: 与 Title 一致
- 内部链接：12 个
- 外部链接：5 个（ASME, API 标准页面）
- Schema: Product + FAQPage

**优势**:
- 技术数据详尽
- 对比表格清晰
- 权威标准引用
- 有原理动画演示

**劣势/差距**:
- 缺少安装指南
- 无价格区间参考
- 缺少 B2B 采购建议
- 未覆盖执行器选型

**我们的机会**:
1. 添加安装步骤详解
2. 提供价格影响因素分析
3. 强化 B2B 采购视角
4. 补充执行器/配件选型指南
5. 添加可下载的选型手册 PDF
```

## 4. 内容差距分析（SKILL-18）

**找出对手未覆盖的主题**:
```
方法:
1. 列出前 5 名竞争对手
2. 提取每篇文章的小标题
3. 找出共同缺失的主题
4. 确定我们可以补充的内容

示例发现:
- 对手 A: 缺少安装扭矩数据
- 对手 B: 没有材料对比详解
- 对手 C: 缺少行业标准解读
- 对手 D: 无采购指南内容
- 对手 E: 缺少故障排除指南

→ 策略：覆盖所有缺失点，创建更全面的内容
```

## 5. 反向链接机会

**分析对手外链来源**:
```
工具：Ahrefs / Semrush / Moz

查找:
- 对手获得外链的页面
- 外链类型（行业目录、协会资源、技术论坛）
- 可复制的外链策略

高价值外链来源:
- 行业协会资源页（ASME, API, MSS）
- 工程参考站点（engineeringtoolbox.com）
- 行业目录（ThomasNet, GlobalSpec）
- 技术论坛推荐（Eng-Tips, CR4）
- 大学/研究机构引用

我们的行动:
1. 创建原创技术数据/图表供他人引用
2. 发布行业标准解读系列
3. 制作可下载的选型手册/计算器
4. 在行业论坛提供专业解答并链接到相关内容
```

## 6. 差异化策略

**B2B 专注**（大多数竞争对手聚焦终端用户）:
- 强调批发/OEM 服务能力
- 添加 MOQ、交货期、认证信息
- 提供样品请求流程
- 展示工厂/生产/质检能力
- 包含项目案例研究

**数据驱动**（原创内容）:
- 材料性能测试数据
- 压力 - 温度额定值图表
- 尺寸重量对比表
- 行业趋势报告
- 失效分析案例

**技术深度**（超越对手）:
- 完整的 ASTM/ASME/API 标准引用
- 详细的材料化学成分表
- 焊接工艺指导
- NDT 检测要求说明
- 更多实际应用案例

**采购支持**:
- 在线询价系统
- 技术规格书下载
- CAD 图纸库
- 材料证书样本
- 质检流程说明
