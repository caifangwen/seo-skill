# SKILL-00A: 关键词调研

**用途**: 写作前进行关键词研究，确定文章 SEO 方向  
**执行时机**: 写作前第一步，在竞对调研（`competitor-research.md`）之前

---

## 1. 关键词类型分类

| 类型 | 说明 | 示例 |
|-----|------|------|
| 核心关键词 | 文章主要目标词 | "best pocket knife steel" |
| 长尾关键词 | 3-5 词的具体查询 | "best steel for EDC knife 2026" |
| 对比关键词 | 包含对比意图 | "S30V vs S35VN", "VG10 vs AUS10" |
| 问题关键词 | 疑问句形式 | "what is magnacut steel", "is D2 steel good" |
| 商业关键词 | 含购买意图 | "buy M390 knife", "wholesale kitchen knives" |

---

## 2. 关键词调研步骤

**步骤 1: 确定主题范围**
```
输入：产品类型/主题
输出：核心关键词列表

示例 - 钢材主题:
- [钢材名] steel
- best steel for [用途]
- [钢材 A] vs [钢材 B]
- what is [钢材名]
- is [钢材名] steel good
```

**步骤 2: 分析搜索意图**
```
信息型 (Informational):
- "what is...", "how to...", "guide to..."
- 适合：科普文章、指南类文章

商业型 (Commercial):
- "best...", "top...", "review..."
- 适合：评测文章、排行榜

交易型 (Transactional):
- "buy...", "wholesale...", "manufacturer..."
- 适合：产品页面、B2B 页面
```

**步骤 3: 关键词难度评估**
```
低难度 (KD 0-30):
- 长尾词、新话题
- 示例："magnacut steel review 2026"

中难度 (KD 30-60):
- 中等竞争词
- 示例："best EDC steel", "VG10 steel properties"

高难度 (KD 60+):
- 核心大词
- 示例："best knife steel", "pocket knife"
```

## 4. 关键词布局规则

**标题 (Title)**: 包含核心关键词，长度 50-60 字符
```
示例："Best Pocket Knife Steels: A Complete Analysis"
```

**开篇段落**: 前 100 词出现核心关键词，自然融入，避免堆砌

**副标题 (H2/H3)**: 每 300-500 词包含 1 个相关关键词，使用长尾变体
```
示例："## Low-end knife steels", "## High-end Knife Steels"
```

**图片 Alt**: 描述性文字 + 关键词
```
示例：alt="pocket knife with red handle one rocks"
```

**内部链接锚文本**: 使用关键词变体
```
示例：[best steel for kitchen knives](https://leeknives.com/best-steel-for-kitchen-knives/)
```
