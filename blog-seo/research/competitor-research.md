# SKILL-00B + SKILL-18: 竞对调研与内容差距分析

**用途**: 分析竞争对手内容，找出差距和机会  
**执行时机**: 关键词调研（`keyword-research.md`）完成后

---

## 1. 竞对识别

**直接竞争对手**:
- 同类刀具制造商网站
- 刀具批发商博客
- 行业权威站点（如 knifesteelnerds.com）

**间接竞争对手**:
- 刀具评测 YouTube 频道
- 刀具论坛（如 BladeForums）
- Reddit 相关板块（r/knives）

---

## 2. 竞对分析维度

**内容结构分析**:
```
□ 文章长度（字数）
□ 章节数量
□ 图片数量/类型
□ 表格使用情况
□ 视频嵌入
□ FAQ 章节
□ CTA 位置和形式
```

**SEO 元素分析**:
```
□ Title 格式和长度
□ Meta Description
□ H1/H2/H3 结构
□ 内部链接数量
□ 外部链接来源
□ 关键词密度
```

**用户体验分析**:
```
□ 页面加载速度
□ 移动端适配
□ 可读性（段落长度）
□ 导航清晰度
□ 视觉设计
```

---

## 3. 竞对分析模板

```markdown
## 竞争对手分析报告

**目标关键词**: best pocket knife steels

### 竞争对手 1: [网站名]
URL: [链接]

**内容分析**:
- 字数：约 3,500 词
- 章节：8 个 H2, 15 个 H3
- 图片：12 张
- 表格：2 个对比表
- CTA: 文末 + 文中 2 处

**SEO 分析**:
- Title: "10 Best Pocket Knife Steels [2026]"（45 字符）
- H1: 与 Title 一致
- 内部链接：8 个
- 外部链接：3 个（权威来源）

**优势**:
- 内容详尽，覆盖全面
- 对比表格清晰
- 权威外链支持

**劣势/差距**:
- 缺少 FAQ 章节
- 图片质量一般
- CTA 不够明显
- 未覆盖 [某钢材]

**我们的机会**:
1. 添加 FAQ 章节捕获精选摘要
2. 使用更高质量图片
3. 添加更多对比数据
4. 覆盖更新/更全面的钢材列表
5. 强化 B2B 导向 CTA
```

## 4. 内容差距分析（SKILL-18）

**找出对手未覆盖的主题**:
```
方法:
1. 列出前 5 名竞争对手
2. 提取每篇文章的小标题
3. 找出共同缺失的主题
4. 确定我们可以补充的内容

示例发现:
- 对手 A: 缺少价格分析
- 对手 B: 没有维护指南
- 对手 C: 缺少 B2B 视角
- 对手 D: 无视频内容
- 对手 E: 缺少最新钢材（如 MagnaCut）

→ 策略：覆盖所有缺失点
```

---

## 5. 反向链接机会

**分析对手外链来源**:
```
工具：Ahrefs / Semrush / Moz

查找:
- 对手获得外链的页面
- 外链类型（客座文章、资源页、论坛）
- 可复制的外链策略

示例:
- knifesteelnerds.com 被大量论坛引用
- 原因：提供钢材测试数据
- 我们的行动：创建原创数据/图表
```

## 6. LeeKnives 差异化策略

**B2B 专注**（大多数竞争对手聚焦 B2C）:
- 强调批发 / OEM 服务
- 添加 MOQ、价格区间信息
- 提供样品请求流程
- 展示工厂/生产能力

**数据驱动**（原创内容）:
- 钢材性能测试数据
- 客户案例研究
- 行业趋势报告
- 对比图表/信息图

**内容深度**（超越对手）:
- 更详细的钢材成分表
- 更多实际使用场景
- 更全面的对比维度
- 更新的行业信息
