# 品牌 SEO 规范细则

## 一、品牌词使用规范

### 1.1 公司名称

**全称：** [公司全称]  
**简称：** [公司简称]  
**英文：** [Company Name]

**使用规则：**
- 首次提及：必须使用全称，并在括号内注明简称
- 后续提及：可使用简称
- 标题中：可使用简称（节省字符）

**示例：**
```
✅ 正确：[公司全称]（以下简称"[公司简称]"）成立于 2020 年...
✅ 正确：[公司简称] 推出了新产品...
❌ 错误：我们公司...（不正式）
```

---

### 1.2 产品名称

**产品 A：**
- 正确：`ProductA`（首字母大写，无空格）
- 错误：`producta`, `PRODUCTA`, `Product A`

**产品 B：**
- 正确：`Product B`（首字母大写，有空格）
- 错误：`product b`, `PRODUCT B`

**版本号：**
- 正确：`v1.0`, `Version 2.0`, `2.0 版本`
- 错误：`V1.0`, `version2.0`, `v 1.0`

---

### 1.3 功能/特性名称

**命名规则：**
- 中文功能名：使用中文引号「」或双引号""
- 英文功能名：首字母大写

**示例：**
```
✅ 正确：使用我们的「智能分析」功能
✅ 正确：使用 Smart Analyze 功能
❌ 错误：使用智能分析功能（无引号）
```

## 二、语调风格指南

### 2.1 目标受众适配

| 受众类型 | 语调 | 术语使用 | 示例 |
|----------|------|----------|------|
| 企业客户 | 专业、正式 | 可使用行业术语 | "提升企业运营效率" |
| 个人用户 | 友好、亲切 | 避免专业术语 | "让您的工作更轻松" |
| 开发者 | 技术、直接 | 使用技术术语 | "API 响应时间<100ms" |

---

### 2.2 禁止用语

**绝对化表述：**
- ❌ 最好、第一、领先、顶级、极致
- ✅ 替代：领先行业、广受好评、深受欢迎

**无法证实的声称：**
- ❌ 保证、100%、绝对、永远
- ✅ 替代：通常、大多数情况下、数据显示

**贬低竞品：**
- ❌ 某竞品问题很多、不如我们
- ✅ 替代：与竞品相比，我们的特点是...

---

### 2.3 推荐用语

**数据支撑：**
```
❌ 我们的工具效率很高
✅ 使用我们的工具，用户平均节省 30% 时间
```

**客户证言：**
```
❌ 用户都很喜欢我们的产品
✅ "这款产品改变了我们的工作方式"——[客户名]，[职位]
```

**场景化描述：**
```
❌ 功能强大
✅ 当您需要在 5 分钟内完成周报时，一键生成帮您节省时间
```

## 三、合规声明模板

### 3.1 数据/效果声明

```
* 以上数据来源于内部测试，实际效果可能因使用情况而异。
* 测试结果基于 [样本数量] 个用户，时间周期为 [时间段]。
```

### 3.2 竞品对比声明

```
* 竞品数据来源于公开渠道（官网、第三方评测），截至 [日期]。
* 对比仅供参考，实际选择请根据具体需求。
```

### 3.3 客户案例声明

```
* 本案例已获得客户授权发布。
* 客户公司名称已做匿名处理。
```

### 3.4 前瞻性声明

```
* 本文包含前瞻性陈述，实际产品功能可能有所调整。
* 功能发布时间仅供参考，请以官方公告为准。
```

## 四、视觉元素规范

### 4.1 图片 Alt 文本

**格式：**
```
alt="[图片内容描述] | [公司/产品名]"
```

**示例：**
```
✅ 正确：alt="ProductA 仪表盘界面 | 公司简称"
❌ 错误：alt="图片 1", alt="截图"
```

---

### 4.2 图表标注

**必需元素：**
- 图表标题
- 坐标轴标签（含单位）
- 数据来源
- 制作日期

**示例：**
```
图 1:2024 年自然流量增长趋势
数据来源：Google Analytics，2024 年 12 月
```

---

### 4.3 截图规范

**必需标注：**
- 产品版本号
- 截图日期
- 关键区域标注（如有）

**示例：**
```
ProductA v2.3 界面截图（2024 年 12 月）
```

## 六、常见问题

**Q：可以在标题中使用竞品词吗？**

A：可以，但需遵循：
- 保持客观中立
- 不误导用户（如"XX 替代品"）
- 内容有实际对比价值

**Q：客户案例可以匿名吗？**

A：可以，但需：
- 获得客户同意
- 标注"已匿名处理"
- 保留真实信息备查

**Q：数据可以估算吗？**

A：可以，但需：
- 明确标注"估算"
- 说明估算方法
- 提供数据范围而非精确值
