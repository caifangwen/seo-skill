## 提交指南

### 准备材料

**公司信息**：
- 公司名称（英文 + 本地语言）
- 公司简介（200-500 词）
- 成立时间、员工数量、年产能
- 出口市场列表
- 联系方式（地址、电话、邮箱、网站）

**产品信息**：
- 产品列表（分类清晰）
- 产品描述（包含关键词）
- 规格范围
- 认证列表
- 产品图片（高质量）

**认证文件**：
- ISO9001 证书
- CE 证书
- Mill Test Certificate 样本
- 其他行业认证

### 提交流程

1. **研究目录**：确认目录相关性和质量
2. **注册账号**：使用公司邮箱注册
3. **填写信息**：完整填写所有字段
4. **上传文件**：上传认证、图片
5. **提交审核**：等待审核通过
6. **定期更新**：每季度更新信息

### 优化技巧

**提高审核通过率**：
- 使用公司邮箱（非 Gmail/Hotmail）
- 填写所有必填和选填字段
- 上传真实、高质量图片
- 提供完整联系方式

**提高曝光效果**：
- 使用关键词丰富的描述
- 上传多张产品图片
- 定期更新产品信息
- 获取客户评价
- 响应询盘及时

## 外链价值评估

### 评估标准

| 标准 | 权重 | 说明 |
|------|------|------|
| 相关性 | 40% | 是否钢铁/B2B 相关 |
| 权威性 | 25% | 域名权威性 |
| 流量 | 20% | 目录本身流量 |
| 难度 | 10% | 获取难度 |
| 成本 | 5% | 费用高低 |

### 优先级排序

**P0（立即提交）**：
- 免费 + 高相关 + 高权威
- 示例：Thomasnet（免费部分）、SteelOrbis

**P1（本周提交）**：
- 免费 + 高相关 + 中权威
- 示例：区域黄页、行业目录

**P2（本月提交）**：
- 付费 + 高相关 + 高权威
- 示例：Alibaba Gold Supplier

**P3（有空时）**：
- 免费 + 中相关 + 低权威
- 示例：小型行业论坛
