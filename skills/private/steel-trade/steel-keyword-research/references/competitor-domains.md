## 竞品分析方法

### 1. 关键词差距分析

使用 `web_search` 搜索：
```
site:competitor.com + [product keyword]
```

示例：
```
site:sail.co.in rebar ASTM A615
site:posco.com HRC supplier
```

### 2. 页面元素分析

检查竞品页面的：
- Title Tag 结构
- Meta Description 写法
- Heading 层级 (H1/H2/H3)
- 规格表格式
- 询盘按钮位置

### 3. 内容覆盖分析

对比竞品覆盖的：
- 产品规格范围
- 标准认证展示
- 应用场景描述
- FAQ 内容

### 4. 外链来源分析

使用 `web_search` 搜索：
```
link:competitor.com OR "competitor name" + steel
```

## 差异化机会

### 针对印度钢厂
- **优势**：交货更快（如果靠近买家）、服务更灵活
- **关键词机会**：强调 "China factory direct", "faster delivery"

### 针对韩国钢厂
- **优势**：价格竞争力
- **关键词机会**：强调 "cost-effective alternative to POSCO"

### 针对土耳其钢厂
- **优势**：价格、产品规格更全
- **关键词机会**：强调 "wider spec range", "competitive price"
