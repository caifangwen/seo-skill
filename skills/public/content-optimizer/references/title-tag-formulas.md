# Title Tag 公式模板

## 基础公式

### 公式 1：标准型
```
[主关键词] - [核心卖点] | [品牌名]
```
**示例：** 在线项目管理软件 - 提升团队效率 50% | ProjectPro

### 公式 2：问题解答型
```
[问题]？[答案概要] - [品牌名]
```
**示例：** 如何选择项目管理软件？2026 年最全指南 - ProjectPro

### 公式 3：数字列表型
```
[数字] + [形容词] + [主题] + [年份/版本]
```
**示例：** 10 款最佳免费在线项目管理工具 2026

### 公式 4：对比型
```
[产品 A] vs [产品 B]：[对比维度] 全面评测
```
**示例：** Trello vs Asana：功能、价格、用户体验全面对比

### 公式 5：教程型
```
如何 [达成目标]：[步骤数] 步完整教程
```
**示例：** 如何用项目管理软件提升效率：7 步完整教程


## A/B 测试建议

为每个页面准备 3 个 Title 版本进行测试：

| 版本 | 策略 | 目标 |
|------|------|------|
| A | 关键词优先 | 排名优化 |
| B | CTR 优先 | 点击率优化 |
| C | 品牌优先 | 品牌认知 |

**测试周期：** 2-4 周
**评估指标：** CTR、排名变化、停留时间
